# Shop Black - Marketplace Online

Um marketplace completo com produtos diversos e lojas parceiras. Encontre tudo que precisa em um só lugar.

## Funcionalidades

- Produtos em destaque por categoria
- Painel de administração para gerenciar produtos
- Design responsivo para mobile e desktop
- Modo escuro/claro
- Integração com plataformas parceiras

## Como usar

1. Abra o arquivo `shop balck.html` em um navegador web
2. Navegue pelas categorias de produtos
3. Para acessar o painel admin, clique em "AD" e use a senha: `M@ria007`

## Deploy Público

Para tornar o site público, você pode hospedá-lo no GitHub Pages:

1. Crie um repositório no GitHub
2. Faça upload de todos os arquivos (HTML, CSS, JS, img/)
3. Vá para Settings > Pages
4. Selecione "Deploy from a branch" e escolha "main"
5. O site ficará disponível em `https://seu-usuario.github.io/nome-do-repo/`

## Estrutura do Projeto

- `shop balck.html` - Página principal
- `styles.css` - Estilos CSS
- `script.js` - Lógica JavaScript
- `img/` - Imagens do projeto
